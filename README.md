switch-translate
================
App allow to switch a set of english letters into readable russian text and vice versa

######Example:
> english-russian example: "Ghbdtn? vbh!" gives 'Привет, мир!'

> russian-english example: "Руддщб цщкдв" gives! 'Hello, world!'
